Подробное описание товара

Шестиструнная электрогитара GIBSON 2019 ES-335 STUDIO EBONY — одна из самых гибких и многофункциональных электрогитар в мире, версия 2019 года.
ES-335 Studio отлично подходит для разных музыкальных направлений и разных стилей игры. За звучание отвечают хамбакеры Classic 57 и динамичная схема
собственной разработки Memphis Tone Circuit Plus (MTC Plus), такое сочетание предоставляет безграничные возможности для творческой реализации.
Корпус гитары выполнен из трех слоев древесины: сочетания клен/тополь/клен. Конструкция - частично полая, с центральным ребром из клена.
Гриф – из одного цельного куска клена, профиля Rounded «C», с накладкой из палисандра. Лады инкрустированы перламутровыми точками. Бридж – фиксированный
ABR-1 слитым цинковым струнодержателем Stopbar. Гитара ES-335 модельного ряда 2019 года представлена в трех вариантах цветового исполнения.
Предлагаемая модель – стильного черного цвета с глянцевой нитроцеллюлозной лакированной отделкой и никелированной фурнитурой.
